Usage

Once you are in canvas page, you might see the canvas
amid and settings below. By hover the cursor over settings
you'ill notice the "upload obj", by pressing one you'ill be able
to load preset models. There are two modes.
1 - free mode
2 - rotatione mode
The free mode is default. To turn rotation mode on hold the RMB.
Normal Mode

Camera's movement:
W:move forward; S:move backward; A:move left; D:move right;
Mouse movement implies camera's view movement. Mouse wheel
is zoom.
Rotation Mode

Whereas you hold Right Mouse Button,
you still able to move camera, but the view is freezing
Now you can spin an object: leftwize-rightwize mouse movement
- along OX upward-downward along OY. Mouse wheel - along OZ.
